# Aplikacja do zarządzania zasobami

System zarządzania zasobami stworzony w technologii ASP.NET Core z wykorzystaniem Blazor WebAssembly. Umożliwia użytkownikom tworzenie, edytowanie, przeglądanie i usuwanie zasobów. Aplikacja zawiera system rejestracji/logowania, uwzględnia uprawnienia użytkowników oraz dostarcza REST API do integracji.

## Główne funkcje
- Rejestracja i logowanie użytkowników z autoryzacją JWT
- Zarządzanie zasobami (CRUD)
- Interfejs webowy oparty o Blazor
- API z dokumentacją Swagger
- Role użytkowników: Admin i Użytkownik
- Testy jednostkowe i UI

## Technologie
- ASP.NET Core 8 (API)
- Blazor WebAssembly (Frontend)
- Entity Framework Core (ORM)
- SQL Server (baza danych)
- JWT (autoryzacja)
- Swagger (dokumentacja API)
- xUnit, Playwright (testy)
