# Instrukcja instalacji

## Wymagania wstępne
- .NET SDK 8.0 lub nowszy
- SQL Server (np. LocalDB)
- Visual Studio 2022 lub VS Code
- Node.js (jeśli używasz Playwright do testów UI)

## Krok po kroku
1. Sklonuj repozytorium.
2. Skonfiguruj połączenie z bazą danych w `appsettings.json`.
3. Zainicjalizuj bazę danych.
4. Uruchom aplikację.
5. (Opcjonalnie) Uruchom frontend Blazor w osobnym projekcie, jeśli nie jest hostowany w tym samym rozwiązaniu.
